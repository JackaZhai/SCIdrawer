# Code of Conduct

## Our Commitment
We are committed to providing a friendly, safe, and welcoming environment for all contributors.

## Expected Behavior
- Be respectful and constructive in discussions and reviews.
- Focus on technical issues, not individuals.
- Accept feedback and resolve disagreements professionally.

## Unacceptable Behavior
- Harassment, insults, discriminatory language, or personal attacks.
- Publishing private information without permission.
- Repeated disruptive behavior in issues, PRs, or discussions.

## Enforcement
Project maintainers may remove, edit, or reject contributions and comments that violate this Code of Conduct.

## Reporting
If you experience or witness unacceptable behavior, report it privately to repository maintainers through a private channel.
